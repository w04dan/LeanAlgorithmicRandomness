# LeanAlgorithmicRandomness

A project formalizing algorithmic randomness, the subfield of computability theory studying objects which appear random to any computable algorithm checking them for properties or patterns, in the proof-verification language Lean.

This project is intended to extend the current support for computability theory in Lean that exists in mathlib.

This project closely follows the development of algorithmic randomness given in Downey & Hirschfeldt's textbook *Algorithmic Randomness and Complexity.*
